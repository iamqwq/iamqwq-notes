# Reflection - 反射

## 1 概述

Java 的反射（ Reflection ）是指在运行时动态地获取一个类的信息，比如获取类的属性、方法、构造器等信息，并可以在运行时动态地调用这些属性、方法、构造器等。Java 反射可以使得 Java 程序在运行时检查和修改其本身的结构、状态和行为，从而实现动态性、灵活性和可扩展性。

在 Java 中有对应表示 " 类 "、" 字段 "、" 方法 " 的类，正是它们的存在使反射成为可能。

1. Class 类：表示一个类，可以获取一个类的基本信息，如类名、类的修饰符、父类、接口等。
2. Field 类：表示一个类拥有的字段，可以通过该类获取和设置一个类的字段。
3. Method 类：表示一个类拥有的方法，可以调用一个类的方法。

反射的特性一般通过以下几个步骤体现：

1. 获取一个类的 Class 对象。
2. 根据 Class 对象获取类的构造器、方法、属性等信息。
3. 根据获取到的构造器、方法、属性等信息，进行相应的操作。

## 2 使用反射

### 2. 1 获取类的 Class 对象

根据在上一节中阐述的那样，一切的开始都从获取一个类的对象开始，获取一个类的对象有以下几种方式：

1. 通过类的静态字段 `class` 获取：

   ```java
   Class cls = Integer.class;
   ```

2. 通过实例变量的 `getClass()` 方法获取：

   ```java
   Integer n = Integer.valueOf(1);
   Class cls = n.getClass;
   ```

3. 通过类的完全限定名（类名全称）来获取：

   ```java
   Class cls = Class.forName("java.lang.Integer");
   ```
